# 系统管理

系统管理模块是系统的基础模块，承担着系统访问控制、用户权限分配、设备运维支撑、数据全生命周期管理及系统运行保障等关键职能。本模块通过**整合注册**、**登录**、**用户管理**、**设备管理**、**危害检测**、**告警管理**、**应急指挥**等子模块，构建一套安全、高效、可扩展的管理体系，为危害识别识别核心业务的稳定运行提供全方位支撑，确保系统操作可追溯、权限可管控、数据可利用、设备可监控。

### 权限

目前系统权限只做按钮的控制（没有权限则禁用按钮或者禁止勾选），不做页面菜单权限、数据权限控制

- 用户权限
  - 新增
  - 编辑
  - 查看
  - 删除
- 告警权限只设置是否可以对告警信息进行操作处理，默认所有用户都能查看所有告警信息
  - 编辑
- 应急指挥权限只设置是否可以对应急事件进行操作处理，默认所有用户都能查看所有告警信息
  - 编辑

:::info管理员账户：添加admin用户，则勾选所有权限字段

:::
